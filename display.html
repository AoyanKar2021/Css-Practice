Display- (Block/Inline/Inline-block/none) Practice

  • Display: inline-block, the top and bottom margins/paddings are respected
  • Display: inline-block allows to set a width and height on the element
  • Display: inline-block does not add a line-break after the element
  • Block Element can access width,height,padding(4 area),Margin(4 area)
  • Inline Element can not access width,height
  • Block Element can access padding(4 area)
  • Margin from top and bottom are not applicable for inline Element
  • Margin from left and right are applicable for inline Element
